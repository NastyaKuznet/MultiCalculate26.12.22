Перевод целых и вещественных чисел для пятиклассника С#
Доброго времени суток.

Вам необходимо разработать обучающий-объясняющий комплекс программ, объясняющий на уровне "пятиклассника" как проходит процесс вычислений 
по теме "представление целых и вещественных чисел".

Необходимо реализовать следующий функционал:

1. Перевод целых чисел в дополнительный код

2. Сложение целых (положительных и отрицательных) чисел с использованием дополнительного кода.

3. Реализовать процесс перевода вещественных чисел в формат с плавающей точкой

4. Реализовать процесс сложения вещественных чисел в формат с плавающей точкой.

Обязательные требования для каждого задания:
1. Обеспечить интуитивно понятный и дружественный интерфейс.
2. Обеспечить защиту "от дурака". Предусмотреть реакцию системы на некорректный ввод со стороны пользователя.
3. Показать не просто ответ, а показать ПРОЦЕСС вычислений.
4. В качестве ответа прислать ОДИН zip файл с проектом C# (или другой язык), а также исполняемый файл (exe). 
5. zip архив должен быть проименован ФАМИЛИЯ_ИО_ГРУППА.zip
